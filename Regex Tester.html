Regex Cheatsheet

\d - Digit (0-9)
\w - Word character (a-z, A-Z, 0-9, _)
\s - Whitespace
. - Any character
^ - Start of string
$ - End of string
[abc] - Any of a, b, or c
[^abc] - Not a, b, or c
a|b - a or b
